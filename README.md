## Encode a Message:
Type your message in the "Encode" text field.
Click the Encode button (green).

---

## Decode a Message:
Paste the encoded text into the "Decode" field.
Click the Decode button (red).

--- 
## Copy Results:
Long-press the encoded/decoded text to copy it.
